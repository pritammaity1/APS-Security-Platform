# APS Security Dashboard UI

## Overview

This project is a responsive cybersecurity dashboard UI built using React and TailwindCSS.

## Tech Stack

- React + Vite
- React Router
- Tailwind CSS
- React Icons

## Features

- Dark / Light Mode toggle
- Sidebar navigation
- Dashboard screen
- Active Scan Detail screen
- Live console simulation
- Finding log with severity badges
- Responsive layout
- Mock data (no backend)

## Known Limitations

- No real backend authentication
- Scan detail page uses static mock data
- Filter/Column panels not yet expanded

## How to Run

- git clone https://github.com/pritammaity1/APS-Security-Platform.git
- cd APS-Security-Platform
- npm install
- npm run dev
